The following steps will guide you through the manual process.

Step 1 - PDRE Clean Install Files

Connect via ssh:

ssh {linuxattuneuser}@{attunenode}

Execute the following script: 

                
rm -rf attune_install

Step 2 - Stop Attune Service

Execute the following script: 

                
sudo systemctl stop attune || true

if pgrep -laf run_attune_server 2>&1 > /dev/null; then
    echo "Killing Attune"
    pkill -9 -f run_attune_server
else
    echo "Attune has stopped"
fi

Step 3 - PDRE Deploy Install Files

Copy the File(s) Attune_Enterprise_Release.tar.bz2 to the target node.

Extract and deploy the File(s) here:

attune_install, relative to the home directory

Step 4 - PDRE Delete Old Releases

Execute the following script: 

                
rm -rf python

Step 5 - PDRE Delete Old Logs

Execute the following script: 

                
echo "Deleting attune log files"
rm -f attune.home/*.log*

Step 6 - PDRE Create virtualenv

Execute the following script: 

                
which virtualenv
if [ "$HOME/opt/bin/virtualenv" == `which virtualenv` ]; then
    echo "SUCCESS, this is the right virtualenv"
else
    echo "This isn't the right virtualenv"
    false
fi

virtualenv ~/python

which python
if [ "$HOME/python/bin/python" == `which python` ]; then
    echo "SUCCESS, this is the right python"
else
    echo "This isn't the right python"
    false
fi

Step 7 - PDRE Install Release

Execute the following script: 

                
if [ "$HOME/python/bin/pip" == `which pip` ]; then
    echo "SUCCESS, this is the right pip"
else
    echo "This isn't the right pip"
    false
fi


cd attune_install/attune_ee*

# Deploy the software
pip install --no-index --no-cache  --find-links=py py/attune-*.whl

This script may require you to answer the following prompts:

Prompt Answer
Do you want to permanently enable this release? y
Do you want to update the init scripts to auto start peek? n

Step 8 - PDRE Install Systemd and Restart

Connect via ssh:

ssh {linuxrootuser}@{attunenode}

Execute the following script: 

                
cd ~{linuxAttuneUser.user}/attune_install/attune_ee*

# Install the init script
sed -i -e 's%#ATTUNE_HOME#%/home/{linuxAttuneUser.user}%' init/attune.service
sudo cp init/attune.service /lib/systemd/system/

# Reload systemctl
sudo systemctl daemon-reload

sudo systemctl enable attune

Step 9 - PDRE Start Attune Service

Connect via ssh:

ssh {linuxattuneuser}@{attunenode}

Execute the following script: 

                
# Check if attune is running
if pgrep -laf run_attune_server 2>&1 > /dev/null; then
    echo "Attune is running, it should already be stopped."
    false
fi

echo "Removing old logs"
[ ! -f ~/attune.home/attune.log ] || rm ~/attune.home/attune.log

echo "Restarting PostgreSQL and Attune"
sudo systemctl restart postgresql-15
sudo systemctl restart attune

# Wiat for Attune to start
sleep 20s

# Check if attune is running
if pgrep -laf run_attune_server 2>&1 > /dev/null; then
    echo "Attune is running, GREAT."
else
    echo "Attune has failed to start" >&2
    tail -n50 ~/attune.home/attune.log
    false
fi
